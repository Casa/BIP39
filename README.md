# BIP39
Swift implementation of [Bitcoin BIP39](https://github.com/bitcoin/bips/blob/master/bip-0039.mediawiki): Mnemonic code for generating deterministic keys.

## Requirements
* Xcode 10.2
* Swift 5.0
